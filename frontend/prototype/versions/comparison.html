5分料理 Three visual directions

Fiveish Meals / fiveish-meals

三个版本放在一起看，选产品气质。

同一套 PWA 功能流：首页选择、抽卡、汇总、做菜指引、5 分钟计时器、菜谱库和安装入口。差异只在视觉系统与品牌语气。

快速判断

如果目标是先做出有记忆点的独立产品，优先看活泼版；如果想更像成熟健康工具，Claude 版更稳；BMW 版适合做风格实验。

Claude：温暖、克制、像认真写给自己的健康助手。 BMW：冷峻、高级、强品牌姿态，但幽默感更少。 活泼：最贴合“低分传说卡”，传播性最好。
Version A

Claude 风格

温暖纸面、柔和分隔、琥珀行动色。把搞笑包进一个认真、可信、低噪声的健康工具里。

适合：真正准备长期做 PWA、想保留轻吐槽但不显得玩票。 风险：抽卡的荒诞感会更克制，需要靠文案和动效补记忆点。
打开 Claude 版 看抽卡
Version B

BMW 风格

深色首屏、白色内容区、锐利零圆角、BMW Blue 交互。像高端配置器在认真安排明天吃什么。

适合：做视觉实验、验证“反差感”：豪华系统装一个 5 分料理产品。 风险：高级感会压住食物的亲近感，和电饭煲 icon 的喜感不完全同频。
打开 BMW 版 看抽卡
Version C

活泼厚边框版

奶油底、橙色按钮、厚边框硬投影、夸张弹跳。最像“抽中低分传说卡”的原始产品方向。

适合：独立 PWA 第一版、社交传播、把“5 分但有用”的荒诞感打出来。 风险：如果后续要做严肃营养管理，需要再收敛一点信息密度和视觉噪声。
打开活泼版 看抽卡
